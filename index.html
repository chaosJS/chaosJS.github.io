babel 相关

  1. 基本

    1. presets/plugins

  2. @babel-polyfill （现以弃用，推荐直接使用 core-js/regerneator ）

    1. polyfill 新语法的补丁/兼容性。是 core-js/regerneator 的集合
    2. core-js/regerneator。 core-js 暂不支持 generator 语法，需要用 regenerator 支持
    3. 按需引入@babel-polyfill

  3. babel-runtime

    1. 配置使用 babel-runtime 在转换期间会做处理，不会污染全局变量
    2. 其他人使用不会有副作用

React 相关

react

  1. event

    1. vue 中 event 是原生，事件挂载到当前元素
    2. react 中 event 是合成事件，e.nativeEvent 是原生事件
    3. 所有事件挂载到 document 上

  2. setState

    1. 不可变特性
    2. 可能异步更新
      • 在组件生命周期或 React 合成事件中，setState 是异步
      • 在 setTimeout 或者原生 dom 事件中，setState 是同步
    3. 多次可能会合并
      • 对象形式 setState({})
      • 函数形式不会合并 setState(()=>{})

  3. 生命周期

  4. refs/非受控组件

  5. portals

  6. context

  7. 异步组件

    1. import()
    2. React.lazy()/suspense

  8. 性能优化

    1. scu
    2. purecomp/React.memo
    3. immutable

  9. HOC/render props

react hook

  1. 函数组件：无实例/this， 无生命周期， 无 state，只有 props

  2. useEffect()

  5. 规范

    1. 只在函数组件和自定义 hook 中使用
    2. 只在顶层代码中使用，不能在循环判断中使用
    3. 函数组件执行完就销毁，无论是初始化还是组件更新， 都会重新执行函数，获取最新的组件。这一点和 class 组件不一样
